# Learn Language Platform

## Запуск проекта

### Backend
```bash
cd backend
python -m venv venv
venv\Scripts\activate  # Windows
pip install -r requirements.txt
python run_backend.py
```
Backend: http://localhost:8000

### Frontend
```bash
cd frontend
npm install
npm start
```
Frontend: http://localhost:3000
